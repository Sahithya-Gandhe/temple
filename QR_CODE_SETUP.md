# QR Code Setup Instructions

## How to Add Your UPI QR Code

1. Generate your UPI QR code from your payment app (Google Pay, PhonePe, Paytm, etc.)
2. Save the QR code image as `qr-code.png` or `qr-code.jpg`
3. Place the file in: `public/assets/qr-code.png`

The website will automatically display your QR code in:
- Home page (Annadanam section)
- Donate page (multiple locations)

### Recommended QR Code Specifications:
- Format: PNG or JPG
- Size: At least 500x500 pixels
- Background: White or transparent
- Clear and scannable
